## Klonování projektu
```bash
git clone https://github.com/mudrhaldaigor73/foto-katerina.git
cd foto-katerina
npm install
npm run dev
```

## Větve
- `main` — produkční větev (automatický deploy na Vercel)
- `dev` — vývojová větev (před mergem do main)

## Stack
- React 19 + TypeScript
- Vite
- Tailwind CSS (utility třídy přes CSS variables)
- Framer Motion
- React Router DOM
- React Hook Form
